# Poland - Sales and Stock


        Bridge module to compute the Sale Date for the invoice when sale_stock is installed
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_pl_sale_stock
```

## Dependencies

This addon depends on:
- l10n_pl
- sale_stock
