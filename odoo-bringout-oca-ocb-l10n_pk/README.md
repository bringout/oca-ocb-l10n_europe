# Pakistan - Accounting


Pakistan Accounting Module
=======================================================
Pakistan accounting basic charts and localization.

Activates:

- Chart of Accounts
- Taxes
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_pk
```

## Dependencies

This addon depends on:
- account
