# Germany - Accounting


Dieses  Modul beinhaltet einen deutschen Kontenrahmen basierend auf dem SKR03.
==============================================================================

German accounting chart and localization.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_de
```

## Dependencies

This addon depends on:
- account
- base_iban
- base_vat
- l10n_din5008
