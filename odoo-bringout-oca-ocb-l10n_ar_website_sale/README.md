# Argentinean eCommerce

Be able to see Identification Type and AFIP Responsibility in ecommerce checkout form.

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_ar_website_sale
```

## Dependencies

This addon depends on:
- website_sale
- l10n_ar
