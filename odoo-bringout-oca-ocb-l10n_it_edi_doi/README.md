# Italy - Declaration of Intent


    Add support for the Declaration of Intent (Dichiarazione di Intento) to the Italian localization.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_it_edi_doi
```

## Dependencies

This addon depends on:
- l10n_it_edi
- sale
